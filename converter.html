가계부 변환기

뱅크샐러드에서 받은 xlsx (가계부 내역 시트 포함)를 업로드하면 월별 시트로 정리해서 다운로드 가능. 양수/음수(환불/취소) 자동 상쇄.

1. 사용자 선택 & 파일 업로드

← 이 파일이 누구 것인지 먼저 선택. MJ와 HJ 파일을 각각 업로드하면 좌/우로 함께 표시됩니다.
📁
여기에 뱅크샐러드 xlsx를 드래그하거나 클릭해서 선택
"가계부 내역" 시트의 컬럼: 날짜 / 시간 / 타입 / 대분류 / 소분류 / 내용 / 금액 / 화폐 / 결제수단 / 메모

2. 처리 옵션

기본값: '지출' 타입만, 자동 상쇄 ON. 옵션 변경 후 별도 갱신 없이 즉시 화면 반영.

3. 월별 거래 내역 가맹점명·카테고리 셀 클릭하면 직접 수정 가능

위에서 파일을 업로드하면 여기에 표시됩니다

4. 내보내기

xlsx 파일: 월마다 한 시트씩 이미지 양식(MJ 좌 / HJ 우).

5. 구글시트 동기화 (선택)

Apps Script 웹앱 URL을 한 번 등록해두면 클릭 한 번으로 구글시트의 월별 탭이 갱신돼요. 설정 방법 보기 ▼

일회성 설정 (5분):
  1. 새 구글시트를 만들고 → 메뉴 확장 프로그램 → Apps Script.
  2. 같은 폴더의 apps_script.gs 파일 내용을 통째로 복사해 Apps Script 편집기에 붙여넣고 저장.
  3. 오른쪽 위 배포 → 새 배포 클릭. 유형은 웹 앱.
    • 실행: 본인 (내 계정)
    • 액세스: 본인만 (또는 본인이 필요한 범위)
  4. 처음 한 번은 권한 승인 절차가 떠요. "고급 → 안전하지 않은 페이지로 이동" → 계정 선택 → 허용.
  5. 배포 완료 후 나오는 웹 앱 URL을 복사해 아래 입력란에 붙여넣고 저장.
※ 브라우저 CORS 정책상 응답은 읽지 못해요(opaque). "동기화 성공" 메시지가 떠도 정확한 결과는 구글시트를 직접 새로고침해서 확인하세요.
📄 구글시트 열기
MJ/HJ는 각각 별도 시트로 저장됩니다 (예: MJ2026-05, HJ2026-05). 2026년 데이터만 동기화. 이미 시트에 있는 거래는 건드리지 않고 새 거래만 끝에 추가. 매칭 키: 날짜·이름(B)·금액(D). C열은 임의 수정 가능 (보존됨).
