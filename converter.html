현대카드 · 국민카드 · 하나카드 명세서(.xls / .xlsx)를 업로드하면 거래일자 / 가맹점명 / 금액 / 카테고리로 정리됩니다.

1. 사용자 선택 & 파일 업로드

← 업로드할 파일이 누구 카드인지 먼저 선택
📁
여기에 카드 명세서(.xls / .xlsx)를 드래그하거나 클릭해서 선택
여러 파일 한 번에 가능 · 카드사는 자동 감지 (현대 / 국민 / 하나)

2. 쿠팡 주문 데이터 (선택사항)

국민카드 명세서엔 "쿠팡(쿠페이)" 같은 PG 이름만 찍혀요. 쿠팡 주문 시트의 5월 데이터를 여기에 붙여넣으면 날짜+금액 매칭으로 상품명을 자동 채워줍니다.
형식: 날짜,상품명,가격(원),카테고리,개인화태그,쇼핑몰,주문번호 (헤더 포함 CSV)

3. 거래 내역 가맹점명·카테고리 셀을 클릭하면 직접 수정 가능

위에서 파일을 업로드하면 여기에 표시됩니다

4. 내보내기

xlsx 파일에는 메인 시트(이미지 양식) + 카테고리요약 + 카드별합계 3개 시트가 들어가요.
